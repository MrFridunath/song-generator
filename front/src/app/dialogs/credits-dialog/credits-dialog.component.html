App web creada para el hackatón organizado por Vercel y MiduDev. Web app created for the hackathon organized by Vercel and MiduDev.
El front-end ha sido creado usando el framework de Angular.io y Angular Material versión 17. The front-end has been created using the Angular.io framework and Angular Material version 17.
Para el back-end, se utiliza una API creada por el usuario de GitHub no-se-quien, que permite usar los servicios de la plataforma Suno, de creación de música. For the back-end, an API created by GitHub user no-know-who is used, which allows you to use the services of the Suno music creation platform.
También se utiliza un servidor y una API creados con Express.js para NodeJS, donde usamos el SDK de Vercel para hacer las generaciones de texto por IA. A server and API created with Express.js for NodeJS are also used, where we use the Vercel SDK to do the AI text generations.
He dado la posibilidad de utilizar 4 modelos diferentes, para que puedas probar la eficacia o diferencia entre estos modelos. Con API Keys propias exclusivas para este proyecto, pero también puedes utilizar las tuyas si quieres, o ves que falla alguna de las mías. I have given the possibility of using 4 different models, so you can test the effectiveness or difference between these models.
